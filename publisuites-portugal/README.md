# Publisuites Portugal Lead Sourcing

Goal: collect Portuguese websites for outreach.

## Target fields
- web_url
- topic
- contact_email
- has_email
- contact_method

## Category taxonomy
Use only these fixed categories:
- Business
- Tech
- Lifestyle
- Travel
- Finance
- Cooking
- Sports

## Notes
- Focus: Portuguese content-publishing websites in the target verticals above
- Prefer official website contact pages for emails
- If no email is found, mark `has_email=no`
- Avoid duplicates
- Local-first workflow; Google Sheets sync later
